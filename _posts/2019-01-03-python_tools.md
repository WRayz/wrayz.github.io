 運作原理

```txt
Python => CPython => Python Byte Code => Python Virtual Machine => Machine Code
```

使用其他 Implementation，可以使用不同語言的  library，使用 Jython 舉例運作原理

```txt
Python => Jython => Java Byte Code => Java Virtual Machine => Machine Code
```
